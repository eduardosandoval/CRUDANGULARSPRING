-----------------------------------------------------

- La Base de datos que se utilizo en el desarrollo es postgresql version 9.3 el cual se encuentra adjunta en el archivo comprimido

- Proyecto back realizado en spring boot con java 8 , en el archivo application.properties se encuentran los datos de configuracion para conectarse con la base de datos

- Front realizado en angular version 11

------------------------------------------------------

La aplicacion permite registrar los datos de los cocineros, clientes, 

camareros, las mesas con su respectiva ubicacion , la creacion de facturas

de los platos consumidos y realizar diferentes reportes.

En el despliegue inicial de la aplicacion se  abre una pantalla de logueo el cual no esta funcionando
 
, por favor  ingresar cualquier usuario , despues de realizar el logueo se encontrara con un menu a
 
el lado izquierdo de la aplicacion , desde ahi se podra dirigir a las diferentes pantallas de la aplicacion para comenzar a trabajar.

Se realizaron pruebas antes de enviar la prueba con el fin de que tengan un ambieente optimo para trabajar.
